Project 2:
The Premium Club Multi-Page Portal
Project Description
In this project, you will develop a functional three-page web application that serves as a professional entry point (The Hub), second page your Resume and a specialized retail environment (The Apple Store).
You will start by building index.html, a minimalist landing page designed with Flexbox to achieve perfect symmetry and "Glassmorphism" styling. From there, you will construct apple.html, an interactive storefront that utilizes CSS Grid to organize product listings. This project focuses on the "user journey"—ensuring a visitor can navigate between a high-energy home screen and a clean, structured shop while experiencing smooth, cinematic animations. The third page should open up your resume with CSS
 
Learning Objectives
By the completion of this project, you will be able to:
•	Master 2D Layouts: Implement CSS Grid (repeat(3, 1fr)) to create a responsive 3-column gallery that handles content alignment automatically.
•	Execute Perfect Centering: Use Flexbox (justify-content and align-items) to center UI elements precisely in the middle of the viewport.
•	Apply Advanced Positioning: Utilize position: sticky to keep navigation menus accessible at the top of the page during long scrolls.
•	Integrate Chapter 9 Interactivity: Create a "live" feel using transition and transform to trigger scale and lift effects when a user hovers over elements.
•	Maintain Design Consistency: Use global resets (box-sizing) and shared color palettes (wheat, whitesmoke) to ensure a professional, cohesive brand identity across multiple files.
